# Changelog

## 0.2.0

- 新增 `PDF 默认页面大小` 设置，支持 `实际大小`、`适合页面`、`适合宽度` 与自定义百分比
- PDF 页面大小设置仅对首次在 Zotero 内置 PDF 阅读器中打开的文档生效
- 设置页重构为暖色调卡片布局，随机阅读与 PDF 设置分区展示
- 文件夹选择弹窗统一为暖色调视觉风格
- 补充 README、中文说明与开发文档

## 0.1.0

- 首次发布 `Zotero Random Read`
- 新增顶部工具栏骰子按钮
- 支持从 Zotero 文件夹白名单中随机打开附件
- 支持树状多选文件夹选择器
- 支持本地阅读历史记录与“清除阅读数据”
- 输出构建包 `.scaffold/build/zotero-random-read.xpi`
